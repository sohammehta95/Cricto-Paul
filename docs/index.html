About Project




This project aims to find whether it is possible to know the outcome of a cricket match and predict a winner based on past data and given features. It aims to predict the result of a future match similar to Octopus Paul but in a much informed and sophisticated manner. The project also deals in understanding the data of all the cricket matches ever played and find relevant insights in them.

This project is developed by Soham Mehta, Sai Prasanth Gumpalli and Venkata Kedarnath Pakala under the guidance of Prof. Steven Skiena at the Stony Brook University.
